# Waste Collection Calendar – How to Subscribe

You can use this by subscribing to the private “Waste collection” calendar.
The setup depends on the device and calendar app you use.

## Google / Gmail users (Android & Google Calendar)

1. Send Dimitris a message with your Gmail address
   (for example: joe.doe@gmail.com)

2. You’ll be added to the calendar

3. The events will appear immediately in Google Calendar and on your phone

That’s it — no further setup needed.

## Apple users (iPhone & Mac)

### iPhone

1. Open **Calendar**

2. Tap Calendars → Add Calendar → Add Subscription Calendar

3. Paste this link:

```bash
https://calendar.google.com/calendar/ical/7u5r90lguj62vq7hd4p95fkci4%40group.calendar.google.com/private-1dada991087dc1be52f5f118f09e63ed/basic.ics
```

3.⁠ Tap **Subscribe**

### Mac

1. Open **Calendar**

2. Go to File → New Calendar Subscription…

3. Paste this link:

```bash
https://calendar.google.com/calendar/ical/7u5r90lguj62vq7hd4p95fkci4%40group.calendar.google.com/private-1dada991087dc1be52f5f118f09e63ed/basic.ics
```

3.⁠ ⁠Click **Subscribe**
